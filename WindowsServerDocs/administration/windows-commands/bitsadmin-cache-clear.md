# <a name="bitsadmin-cache-and-clear"></a>bitsadmin cache and clear

Löscht den lokalen Cache.

## <a name="syntax"></a>Syntax

```
bitsadmin /cache /clear
```

## <a name="examples"></a>Beispiele

So löschen Sie den lokalen Cache:

```
bitsadmin /cache /clear
```
