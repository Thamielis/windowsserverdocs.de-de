# <a name="bitsadmin-peers-and-discover"></a>bitsadmin peers and discover

Ermittelt Peers erneut.

## <a name="syntax"></a>Syntax

```
bitsadmin /peers /discover
```

## <a name="examples"></a>Beispiele

So ermitteln Sie Peers erneut:

```
bitsadmin /peers /discover
```
