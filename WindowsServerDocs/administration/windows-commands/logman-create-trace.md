# <a name="logman-create-trace"></a>logman Create Trace

> Gilt für: Windows Server (halbjährlicher Kanal), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Erstellen Sie einen Ereignis Ablauf Verfolgungs Datensammler.  

## <a name="syntax"></a>Syntax  
```  
logman create trace <[-n] <name>> [options]  
```

## <a name="remarks"></a>Bemerkungen  
Wenn [-] aufgeführt ist, wird die-Option durch ein extra negiert.  
## <a name="examples"></a>Beispiele  
Um einen Ereignis Ablauf Verfolgungs Daten-Collector mit dem Namen trace_log zu erstellen, der nicht weniger als 16 und nicht mehr als 256 Puffer verwendet, wird jeder Puffer mit einer Größe von 64 KB erstellt, und die Ergebnisse werden an den Speicherort c:\logfile ausgegeben.  
```  
logman create trace trace_log -nb 16 256 -bs 64 -o c:\logfile  
```
