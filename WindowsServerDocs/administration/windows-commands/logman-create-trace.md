# <a name="logman-create-trace"></a>logman Create Trace

>Gilt für: Windows Server (Semi-Annual Channel), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Erstellen Sie einen Ereignis Ablauf Verfolgungs Datensammler.  

## <a name="syntax"></a>Syntax  
```  
logman create trace <[-n] <name>> [options]  
```

## <a name="remarks"></a>Hinweise  
Wenn [-] aufgeführt ist, wird die-Option durch ein extra negiert.  
## <a name="examples"></a><a name=BKMK_examples></a>Beispiele  
Im folgenden Beispiel wird ein Ereignis Ablauf Verfolgungs Daten-Collector namens trace_log erstellt, der nicht weniger als 16 und nicht mehr als 256 Puffer verwendet, wobei jeder Puffer 64 KB groß ist, und die Ergebnisse werden an den Speicherort c:\logfile ausgegeben.  
```  
logman create trace trace_log -nb 16 256 -bs 64 -o c:\logfile  
```
