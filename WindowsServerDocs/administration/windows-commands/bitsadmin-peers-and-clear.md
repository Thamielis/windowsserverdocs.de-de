# <a name="bitsadmin-peers-and-clear"></a>bitsadmin peers and clear

Löscht die Peerliste.

## <a name="syntax"></a>Syntax

```
bitsadmin /peers /clear
```

## <a name="examples"></a>Beispiele

, Wenn die Peer Liste gelöscht werden soll.

```
bitsadmin /peers /clear
```
