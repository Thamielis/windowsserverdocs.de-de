# <a name="bitsadmin-peers-and-list"></a>bitsadmin peers and list

Listet alle Peers auf.

## <a name="syntax"></a>Syntax

```
bitsadmin /peers /list
```

## <a name="examples"></a>Beispiele

Auflisten aller Peers:

```
bitsadmin /peers /list
```
