# <a name="rescan"></a>Neu einlesen

Mit dem DiskPart-Befehls Interpreter können Sie neue Datenträger suchen, die Ihrem Computer hinzugefügt wurden.

## <a name="syntax"></a>Syntax

```
rescan
```
