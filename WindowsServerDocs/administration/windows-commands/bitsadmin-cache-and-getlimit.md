# <a name="bitsadmin-cache-and-getlimit"></a>bitsadmin cache and getlimit

Ruft das Cache Limit ab.

## <a name="syntax"></a>Syntax

```
bitsadmin /cache /getlimit
```

## <a name="examples"></a>Beispiele

So rufen Sie das Cache Limit ab:

```
bitsadmin /cache /getlimit
```
