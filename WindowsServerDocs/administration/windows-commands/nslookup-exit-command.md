Beendet das Befehlszeilen Tool Nslookup.

## <a name="syntax"></a>Syntax

```
nslookup /exit
```

| Parameter | Beschreibung |
| --------- | ----------- |
| /? | Zeigt die Hilfe an der Eingabeaufforderung an. |
| /help | Zeigt die Hilfe an der Eingabeaufforderung an. |
