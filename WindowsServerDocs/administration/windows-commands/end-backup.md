# <a name="end-backup"></a>Sicherung beenden

Beendet eine vollständige Sicherungs Sitzung und gibt ggf. ein **BackupComplete** -Ereignis mit dem entsprechenden Writer-Status aus.

## <a name="syntax"></a>Syntax

```
end backup
```
